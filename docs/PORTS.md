# Порты, которые нужно открыть наружу

| Порт | Протокол | Назначение |
|---|---|---|
| 80, 443 | TCP | HTTP/HTTPS — всё веб + федерация Matrix на 443 |
| 7882 | UDP | LiveKit ICE/UDP — основное медиа звонков |
| 7881 | TCP | LiveKit ICE/TCP — фоллбэк |
| 3478 | UDP+TCP | coturn / TURN |
| 5349 | TCP | TURN over TLS (если используете) |
| 49152–49172 | UDP+TCP | coturn relay range (дефолт; настраивается) |
| 30000–30020 | UDP | LiveKit/TURN relay range (если включён) |

- Один сервер с публичным IP: порты просто должны быть не закрыты фаерволом.
- За NAT/роутером: пробросьте перечисленные порты на сервер с Matrix.
- **Медиа звонков идёт напрямую на сервер**, его НЕЛЬЗЯ проксировать через HTTP-reverse-proxy (через прокси идут только 80/443).
- Точные диапазоны зависят от настроек ролей LiveKit/coturn — смотрите их defaults в апстрим-плейбуке.
